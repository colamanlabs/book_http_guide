### 3.2 메시지의 각 부분

```
메시지는 시작줄, 헤더블록, 본문 이렇게 세 부분으로 이루어진다.
```


|시작줄|HTTP/1.0 200 OK|
|-|-|
|헤더| Content-type: text/plain<br>Content-length: 19|
|본문| Hi! I'm a message!

```
각 줄은 캐리지리턴(ASCII 13) 과 개행문자(ASCII 10) 로 구성된 두글자의 줄바꿈 문자열로 끝난다.
이 줄바꿈 문자열은 "CRLF" 라고 쓴다. 
```
